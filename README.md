### Usage
command-line

    cd endymion/

:

    lua chromia.lua

:

    lua chromia.lua group yq

:

    lua chromia.lua query ^%a%dh?$

:

    lua chromia.lua n0 j3

:

    lua chromia.lua gamut

:

    lua chromia.lua help
